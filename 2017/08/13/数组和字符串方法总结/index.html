数组和字符串方法总结

发表于 2017-08-13

##数组的方法总结

  1. push 往数组末尾追加元素返回值是新数组的长度 arr.push()
  2. pop 从数组的末尾删除一个元素返回值是被删除的那个元素
  3. unshift 往数组的首位插入一个元素返回数组的新长度
  4. shift从数组的首位删除一个元素返回的是被删除的那个元素

arr.indexOf(a,b)查询a元素在数组中的索引号，如果b没有指定那么默认从数组的首位开始查询

a.concat(b) 将a数组和b进行连接成一个新的数组

arr.join() 将数组转化成字符串，如果括号内没有标识，那么生成的字符串是以每个数组元素用逗号进行分隔，如果传入的是一个””，则是将整个数组转化成一个整的字符串。

var arr = [1,2,3,4,5] arr.join() => 1,2,3,4,5

sort()
对数组元素进行排序，但是排序是根据ASC码进行排序的，如果想要准确的排序可以传入一个给sort()传一个回调函数 

var compare(a,b){return a-b;}       arr.sort(compare);

slice(a,b) 表示从a索引位置开始截取 截取到b索引位置，b索引的元素截取不到。如果b没有值，则默认截取到最后。该方法不会改变原数组的值。

splice(a,b,c) 截取数组 如果传2个值，则表示从a位置开始截取b个元素(包括a在内)。会改变原数组的内容，如果有第三个参数，那么被截取掉的部分将会被替换成第三个参数的值。

比如: var arr=[1,2,3,4];arr.splice(1,2,5,6) => arr=[1,5,6,4]; 从数组第二个元素开始剪切掉2 3,替换成 5 6

##字符串的方法总结

str.charAt(index)
传入一个索引号返回该索引号对应的值

str.indexOf(a,b)
一般传一个参数，表示查找a在字符串str中的索引号，如果a是一个连续的字符那么会返回匹配这个字符的首字符的索引。这个特性可以用来解决百度搜索框的案例。

substr(a,b)
截取字符串 从a索引位置开始截取包括a在内的b个字符，不会改变原字符串。

substring(a,b)、slice(a,b)
这两个方法都可以截取字符串，表示从a索引位置开始，截取到b索引位置，b截取不到。不会改变原数组的长度。

split(分隔符)
传入一个分隔符，将这个字符串转化成数组，如果不传参数的话，默认是将整个字符串转换成数组的第一个元素。如果传入的是””那么会将该字符串的每个字符作为一个数组元素进行分隔。

trim()
该方法可以去除字符串左右两端的空格。
